# Kế hoạch cập nhật (Plans)

Thư mục này lưu các kế hoạch thay đổi theo từng hạng mục, để có thể mở rộng nhiều kế hoạch về sau.

## Quy ước đặt tên

- Định dạng file: `YYYY-MM-DD-<slug>.md`
- Ví dụ: `2026-03-25-theme-toggle-below-avatar.md`

## Trạng thái đề xuất

- `draft`: đang lên ý tưởng
- `approved`: đã chốt, sẵn sàng triển khai
- `in-progress`: đang thực hiện
- `done`: đã hoàn thành
- `blocked`: đang bị chặn

## Mẫu nhanh

```md
# <Tiêu đề>

- **ID:** PLAN-XXX
- **Ngày tạo:** YYYY-MM-DD
- **Trạng thái:** draft
- **Phạm vi:** UI / UX / Performance / ...

## Mục tiêu

## Vị trí ảnh hưởng

## Cách triển khai đề xuất

## Checklist

## Rủi ro & cách giảm thiểu

## Tiêu chí hoàn thành
```
